Modules

This section introduces the functionality of the Tatool Web platform in order to get you started quickly. Depending on your assigned user roles, you can see and use the following parts of Tatool:

  • Modules: Add and run Modules. The Modules are either publically available or you've been invited to participate in a private Module.
  • Editor: Create and edit your own Modules with the help of the convenient Module Editor. After you've created your Module, you can easily publish Modules to the Tatool repository for users to run your Modules (only available to researchers).
  • Analytics: Use the Analytics to monitor participation in your experiment with detailed data for every session (only available to researchers).
  • Admin: The Admin can manage users and projects through a convenient user interface (only available in your local installation).
